# Fundamentos python 

En este repositorio estoy guardando todo lo que he aprendido en Python, desde cómo imprimir un mensaje hasta crear lógicas para videojuegos. 

---

## Estructura del Proyecto

- **seccion1**: Mis primeros pasos con la función `print`.
- **seccion2**: Aprendiendo sobre cadenas de texto y caracteres especiales.
- **seccion3**: Dominando las matemáticas y los operadores.
- **seccion4**: Variables, convertidores y algoritmos de juegos.
- **src/puntaje_final_jugador.py**: Un ejemplo práctico de todo lo aprendido.

---

## Resumen de los Laboratorios

### Seccíon 1: Hola Mundo y Formatos
*   **Lab 1**: Mi primer `print("¡Hola, Mundo!")`. Aprendí que si olvidas las comillas o los paréntesis, Python se queja.
*   **Lab 2**: Usé `sep` para separar palabras con símbolos (como asteriscos `***`) y `end` para decidir cómo termina una línea.
*   **Lab 3**: Dibujé una flecha con asteriscos. Aprendí a usar `\n` para saltar de línea y que puedo multiplicar texto (ej. `"*" * 5`).

### Sección 2: Cadenas y Literales
*   **Lab 4**: El reto de imprimir comillas dentro de comillas usando `\"`. Aprendí que para que Python no se confunda, usamos la barra invertida `\` como un "escape".

### Sección 3: Operadores Matemáticos
*   **Lab 5**: Resolví 15 ejercicios matemáticos. Aquí aprendí el orden de importancia:
    1.  **Paréntesis `()`** (Lo que manda).
    2.  **Potencias `**`** (Elevar números).
    3.  **Multiplicación y División `* /`**.
    4.  **Suma y Resta `+ -`**.
    *   *Ejemplo:* `5 + 3 * 2` da **11**, porque la multiplicación se hace primero.

### Sección 4: Variables y Algoritmos
*   **Lab 6**: Creé un convertidor de Millas a Kilómetros usando variables y la función `round()` para que los decimales se vean bonitos.
*   **Lab 7**: Pasé una fórmula matemática compleja (`y = 3x³ - 2x² + 3x - 1`) a código Python.
*   **Lab 8**: ¡El rol de desarrollador de juegos! Hice algoritmos para calcular:
    - Puntaje total de niveles.
    - Vida restante de un personaje en porcentaje.
    - Daño crítico y tiempo de juego.
