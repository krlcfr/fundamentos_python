### Ejercicio 1

**Expresión:**

`5 + 3 * 2`

- **Pregunta:** ¿Cuál es el resultado? ¿Por qué?

### Solucion y Explicacion

**Expresión:**

```python
5 + 3 * 2
```

**Paso a paso:**

1. **Identificar las operaciones y su prioridad:**
    - Suma (`+`)
    - Multiplicación ()
    
    La multiplicación tiene mayor prioridad que la suma.
    
2. **Realizar la multiplicación primero:**
    - `3 * 2 = 6`
3. **Realizar la suma:**
    - `5 + 6 = 11`

**Resultado final:**

```python
11
```
